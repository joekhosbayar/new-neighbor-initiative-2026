# new-neighbor-initiative-2026
## Intent
This repository is meant to deliver a guided internship experience where an intern will create a website for the Third Congressional Mongolian Lunar New Year 2026.

## Core Curriculum
1. HTML
2. CSS
3. JavaScript
4. Source Control
5. CI/CD
